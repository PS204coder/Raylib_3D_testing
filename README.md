## Basic 3d using raylib
  A little project where a sphere follows the mouse cursor (almost works perfectly). The sphere has a texture (its just a png that i can put easli on it beacouse raylib is awesome).
  This also features shaders that add lights (the hardest part of this project)


### 3d sphere
It was made using matrixes that i had to learn how to use and how to change them in live time to make the sphere follow my mouse cursor.
The sphere has basic material that i just slap on a png of my choice (You can see some of them that I used in the files)

### Shaders
I did not know how shaders work at all but with the help of my brother and some exaples in the raylib library i copied a basic shader and edited it to my choice.
The shaders create the ambient color and two diffrent colored lights that if the sphere gets far away from they fade away.
Its really basic but helped me to explore the possibilities of raylib and learn some more C.
Also it helped to understand how most of games work these days.
